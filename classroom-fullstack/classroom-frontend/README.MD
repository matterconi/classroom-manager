# AI Recursive Pipeline Dashboard

Refine dashboard for exploring the output of the AI Recursive Pipeline.

The dashboard lets you browse generated components, snippets, theory notes, categories, collections, relationships, demos, and search results produced by the recursive codebase categorization backend.

## Getting Started

## Available Scripts

### Running the development server.

```bash
    npm run dev
```

### Building for production.

```bash
    npm run build
```

### Running the production server.

```bash
    npm run start
```
